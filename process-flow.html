Show process

Show setup and results.

Use this as the quick checklist for setting up a show and carrying it through to archive.

  1. 1
    Create the show

    Add the show date, title, location, prices, and judge assignment from the admin dashboard.

  2. 2
    Collect registrations

    Riders submit their details, horses, classes, camping, yarding, and payment confirmation through the show registration form.

  3. 3
    Set obstacles

    Open obstacle setup from the event detail screen, enter the judge name, and name each obstacle in course order.

  4. 4
    Download scoring cards

    The PDF uses the saved judge, obstacles, riders, horses, classes, and running order for the show.

  5. 5
    Judge and process results

    Scores are entered by class, placings are processed, and completed classes become ready for results publishing.

  6. 6
    Publish and archive

    Once all classes are processed, the event can be published, archived, and included in points and P&L records.

Club process

Club days, clinics, and records.

Use this checklist for the simpler event workflow that does not need show classes or scoring cards.

  1. 1
    Create the event

    Add the club day, clinic, or external event details so it appears on the calendar and registration pages.

  2. 2
    Collect attendee details

    Participants register online, confirm payment, and note day membership requirements where needed.

  3. 3
    Manage attendance

    Use the event detail page to mark attendance, payment status, transfers, or cancellations.

  4. 4
    Record finances

    Add event costs, shared club costs, other income, and membership income into the P&L pages.

  5. 5
    Archive records

    Completed events and financial years are archived so past participation, points, and reporting stay available.

Registration and event data are managed in Supabase.
